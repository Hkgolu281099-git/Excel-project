📊 Excel Sales Dashboard Project
🔍 Overview

Developed an interactive and dynamic Excel Sales Dashboard to analyze business performance and generate actionable insights. The dashboard transforms raw data into meaningful visualizations, enabling efficient decision-making.

🎯 Key Objectives
Designed a structured dataset for sales analysis
Built an interactive dashboard for real-time insights
Automated data aggregation using Excel formulas and Pivot Tables
Enhanced data visualization for better reporting
📂 Dataset Description
<img width="1315" height="545" alt="Annotation 2026-03-24 142852" src="https://github.com/user-attachments/assets/27f40857-7c06-43b5-93c6-28e21e92e569" />

The dataset consists of transactional sales records with the following attributes:

Order ID – Unique identifier for each order
Order Date – Date of transaction
Customer Name – Name of the customer
City, State, Region – Geographic segmentation
Product Category & Product Name – Product classification
Salesperson – Assigned sales executive
Quantity & Price per Unit – Sales metrics
Discount (%) – Applied discount
Total Sales – Final calculated revenue

⚙️ Tools & Technologies
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Transformation
Conditional Formatting
Advanced Excel Formulas (SUMIF, VLOOKUP, IF, etc.)
Slicers & Interactive Filters

📊 Dashboard Highlights
📌 KPI Metrics: Total Sales, Total Quantity, Average Sales

📅 Time-Based Analysis: Monthly & Yearly trends

🌍 Regional Analysis: Performance by region

📦 Category Insights: Sales distribution by product category

🏆 Top Performers: Best products, customers, and salespersons
🎛 Interactive Controls: Slicers for dynamic filtering

📈 Key Achievements
Improved data readability through structured visualization
Reduced manual analysis effort using automated reporting
Enabled quick identification of high-performing segments
Delivered a user-friendly interface for non-technical users

🚀 How to Use
Download or open the Excel file
Navigate to the Dashboard sheet
Use slicers to filter data dynamically
Review KPIs and charts for insights
